
# rncli-expo-ts
Preset for <b>rncli</b>, works with Expo/TS

## Install
```sh
$ npm install -g rncli-expo-ts
```

## How to use
```javascript
// create a file rncli.config.js in your RN/Expo app root
module.exports = {
    preset: 'rncli-expo-ts'
}
```

## Help
```sh
$ rncli generate help # get all help for this preset
```
